**性質**:設計計畫(#15:計畫、非成果;預測力未驗證前皆為假說)。
**前提**:核心股已選定、資料完整且精準(同前二計畫)。
**命題**:取康波之**循環思想**(#9 允許的啟發層)——景氣與股價的循環結構,尺度可為週/月/季/年——
**剔除一切特定週期值**(40-60 年、42 個月、任何「循環長度=X」的硬編皆禁)。
**思想系譜**:承憲章附錄 B「量子金融藍圖→康波」之紀律化(#9/#16:思想可入、數字不回流)。

## 〇、思想萃取:把 40-60 年剝掉,循環思想剩什麼?

1. **循環是常態**——景氣/資金/盈餘/情緒不走直線,以擴張⇄收縮往復;且**多尺度嵌套**
   (庫存循環/資本支出循環/信用循環/情緒循環同時進行,互相疊加)。
2. **位置決定行為**——同一資訊在循環不同相位意義不同;**相位(phase)比水位(level)重要**。
3. **領先-同時-落後結構**——指標對循環有時序角色;領先者轉向=轉折前兆。
4. **轉折有前兆**——減速先於下降(二階導)、背離先於反轉(領先 vs 同時分歧)。
5. **多尺度共振**——週/月/季/年同向則行情放大,互斥則震盪。

> **方法論轉譯**:不假設「循環多長」(禁),改測「**現在在自身循環的哪裡、往哪走、各尺度是否同向**」
> ——相位/位置/歷時/共振/背離全部**由資料自身定義**(rolling 極值、自身歷史),零固定週期。

## 一、#9 合規工具箱(全部 data-driven、無固定週期)

| 工具 | 測什麼 |
|---|---|
| **range-position**(x−rollmin)/(rollmax−rollmin) | 在自身循環的相位(0=谷、1=峰),多尺度並列 |
| **time-since-extreme**(距 rolling 峰/谷天數) | 循環歷時(資料事件定義,非固定期) |
| **drawdown / runup** 深度與持續 | 收縮/擴張段的深度與齡 |
| **momentum 的 momentum**(二階導/減速度) | 轉折前兆 |
| **多尺度同向計分**(5/20/60/120/252 動能符號一致度) | 共振 vs 互斥(連續分數) |
| **背離量**(兩序列動能差/rank-gap 變化) | 領先 vs 同時、價 vs 量、價 vs 流之分歧 |
| **vol 期限結構**(短窗 vol/長窗 vol) | 風險循環相位 |
| (進階,後期)主週期估計:自相關/頻譜**由資料輸出週期** | 週期長度=資料的輸出、非輸入假設;過 #11 五鏡才留 |

視窗一律 5/20/60/120/252 + 月/季/年(calendar 慣例=**觀察尺度**,非「循環長度」宣稱);
月份 one-hot 之季節性 categorical=日曆事實,允;**禁**任何「N 年循環」「sell-in-May」型硬編規則。
